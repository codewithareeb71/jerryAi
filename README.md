# 🤖 Jerry AI Assistant

A powerful AI Voice Assistant for Windows, built with Python. Jerry AI supports voice interaction, automation, web browsing, file management, and AI-powered conversations in a modern desktop environment.

## ✨ Features

* 🎤 Real-Time Voice Assistant
* 🧠 AI-Powered Conversations
* 🌐 Smart Web Search
* 📂 File & Folder Management
* 🔊 Text-to-Speech Responses
* 🎵 Media Controls
* ⚡ Fast and Lightweight
* 🖥️ Windows Desktop Support
* 🔄 Automatic Reconnection System
* 🎨 Modern User Interface

---

## 📋 Requirements

* Python 3.12+ (Recommended)
* Windows 10 / Windows 11
* Microphone Access
* Internet Connection

---

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/codewithareeb71/jerryAi.git
cd jerryAi
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows (PowerShell)**

```powershell
.\venv\Scripts\Activate.ps1
```

**Windows (CMD)**

```cmd
venv\Scripts\activate.bat
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Install Playwright

```bash
playwright install
```

### Run Jerry AI

```bash
python main.py
```
